babel release 6.0.0

  • 希望成为一个javascript的平台化工具 —— minifiers, linters, formatters, syntax highlighters, code completion tools, type checkers, codemod tools, and every other tool to be using the same foundation to do their job better than ever before
  • 模块化 —— 所有的transforms 被重写为插件， 其使用的api被暴露给所有人
  • 选择加载插件机制 —— babel不再默认加载es2015转义机制，需要手动声明要使用的插件。 为了提供更好的api，所有的tansformer现在是完全独立的
